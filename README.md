# Dily(다이어리)
MVVM 패턴 적용 연습을 위해 간단한 다이어리 앱을 만들어 연습해본다.

## 👉🏻 기능 요구사항
- 생성, 조회, 수정, 삭제 기능 적용
- 제목, 글 감정, 현재 날짜를 적용할 수 있음
- 앱을 종료한 후 다시 실행해도 데이터가 그대로 남아있음
   
## 👉🏻 적용사항
- MVVM 패턴 적용하기
